## 2. 모바일 가로/세로 모드 (게임\_계획 §7)

| 항목             | 내용                                      | 구현                                                                   |
| ---------------- | ----------------------------------------- | ---------------------------------------------------------------------- |
| orientation      | any (가로·세로 모두 지원)                 | ✅ manifest `orientation: "any"`                                       |
| 화면 고정 버튼   | 모바일에서만 표시, Screen Orientation API | ✅ `useOrientationLock`, `OrientationLockButton` (768px 이하에서 표시) |
| 전체화면 권장    | lock 시 fullscreen 요청 후 lock           | ✅ lock() 시 requestFullscreen 후 orientation.lock                     |
| fullscreenchange | 전체화면 해제 시 고정 해제 반영           | ✅ fullscreenchange 리스너로 상태 동기화                               |
